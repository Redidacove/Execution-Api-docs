engine_getPayloadV2

Obtains execution payload from payload build process

Parameter Names​

Payload id​

Schema​

title : 8 hex encoded bytes​
type : string​

Example Request​

{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "engine_getPayloadV2",
  "params": [
    "0x0000000038fa5dd"
  ]
}
